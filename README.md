# valorant_agents_app

Aplikacja mobilna stworzona w środowisku Flutter, służąca do przeglądania bazy agentów z gry Valorant. Projekt powstał z wykorzystaniem oficjalnego REST API oraz lokalnej bazy danych do pełnego wsparcia trybu offline.

## Funckjonalność

- REST API: pobieranie aktualnych danych o agentach bezpośrednio z sieci
- Baza Hive: Zapis danych w pamięci telefonu, aplikacja działa bez internetu
- Zabezpieczenie: brak sieci nie psuje wyglądu (bohaterowie i skille automatycznie dostają ikony zastępcze)
